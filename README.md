# 🛒 E-Commerce Website

A responsive and user-friendly e-commerce web application built using HTML, CSS, JavaScript, and Node.js. The platform allows users to browse products, search items, manage their cart, and complete purchases through a streamlined checkout process.

## 🚀 Features

- 🏠 Home page with featured products
- 📂 Product categories
- 🔍 Product search functionality
- 📄 Product detail view
- 🛒 Shopping cart management
- 💳 Checkout system
- ✅ Order confirmation page
- 📞 Contact page
- ℹ️ About page
- 📱 Responsive design for mobile and desktop

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/ecommerce.git
```

2. Navigate to the project directory:

```bash
cd ecommerce
```

3. Install dependencies:

```bash
npm install
```

4. Run the application:

```bash
npm start
```
